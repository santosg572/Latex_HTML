C5. Problemas de Valor Inicial para ODEs: Métodos de Un Paso
Problemas de valor inicial para ecuaciones diferenciales ordinarias (ODEs) ocurren en casi todas la ciencias.

5.1 Ejemplos

Nuestro promer ejemplo es muy trivial: supongamos que necesitamos calcular la integral
\( I = \int_a^b f(t) dt \)
para alguna funcion dada f en [a,b]. Sea \( y(x) = \int_a^x f(t) dt \), entonces se obtiene
\( \frac{dy}{dx} = f(x), y(a)=0 \) Este es un problema de valor inicial para una ecuacion diferencial de primer orden. Resolviendo la ecuación anterior en [a,b], uno obtiene I = y(b)

Ejemplo 2. Resolver la siguiente integral
\( I = \int_0^{infty} J_0(t^2) e^{-t} dt \) conteniendo la funcion de Bessel \( J_0(x^2) \). esta satisface la ecuacion diferencial de segundo orden lineal
\( \frac{d^2y}{dx^2} + (2 - \frac{1}{x} \frac{dy}{dx} + 4x^2 y = 0, x > 0 \)
con condiciones iniciales
y(0) = 1, y'(0) = 0
Pongamos

\( y_1(x) = \int_0^x J_0(t^2) e^{-t} dt, t_2(x) = J_0(x^2), y_3(x) = y'_2(x) \)

y obtenemos

\( \frac{dy_1}{dx} = e^{-x} y_2, y_1(0) = 0, \)
\( \frac{y_2}{dx} = y_3, y_2(0) = 1 \)
\( \frac{dy_3}{dx} = -(2 - \frac{1}{x}) y_3 - 4x^2 y_2, y_3(0) = 9 \) es un problema de valor inicial para un sistema de ecuaciones diferenciales lineales de primer orden.
5.2 Tipos de Ecuaciones Diferenciales

El problema de valor inicial estandard envuelve un sistema de ecuaciones diferenciales de primer orden

\( \frac{dy^i}{dx} = f^i(x, y^1, y^2, ..., y^d), i=1,2, ...d \)

la cual sera resuelta en un intervalo [a,b], dados los valores iniciales

\( y^i(a) = y^i_0. i= 1,2, ..., d \)

5.5 Descripción Local de Metodos de Un Paso

Dados un punto generico \( x \in [a,b], \mathbf{y} \in \mathbb{R}^n \) definimos a single step del metodo de un-paso por

\( \mathbf{y}_{next} = \mathbf{y} + h \mathbf{\phi} (x, \mathbf{y}; h), h > 0 \)

la función \( \phi: [a,b] \times \mathbb{R}^d \times \mathbb{R}_{+} \rightarrow \mathbb{R}^d \)
